# NekoTags

> Permisos de tags: `nekotags.tag.<tag>`

> Variables de papi: `%nekotags_active_tag%` `%nekotags_active_tag_color%` 

**Comandos:**
> /tags
> 
> /tags select `<tag>` 


> **Admin:** `(nekotags.admin)`
>
>
> /tags give `<user>` `<tag>`
>
> /tags remove `<user>` `<tag>`
>
> /tags list `<user>` `<tag>`
